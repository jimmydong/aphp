# APHP - Docker Image of Apache + PHP

## Install

    Get all files
    edit new_aphp.php, change host settings
    ./new_aphp.sh
    place your apache conf in: /WORK/HTML/apache_conf
    place your website in: /WORK/HTML/***
    docker stop aphp
    docker start aphp

## Tools

    Enter the shell of aphp:
    ./enter_aphp.sh
    
.
